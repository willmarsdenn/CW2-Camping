# CW2-Camping Project

## Camping project

This will use the [AcuWeather API](https://developer.accuweather.com) and the [Geoapify Places](https://apidocs.geoapify.com/docs/places/#categories) to help users find campsites based on location with good weather conditions. **(UK Only)**
